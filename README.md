# MGD_ML
A testing space for improving Machine Learning (ML) predictions on Molecular Glue Degrader (MGD) potencies and selectivity.

## CHEM269_project
Subfolder containing the final project (The Molecular Glue Problem) for UCSD CHEM269 as directed by Dr. Adrian Jinich. All necessary inputs are included here, as well as example outputs for inspection. Make sure to read the **ENTIRE** README.md!

## zbtb11_mgd
Subfolder containing ML pipelines pertaining to the optimization and design of ZBTB11 MGDs.
